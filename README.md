# arbretreee
Description du projet :
Ce projet est une application développée en Java avec JavaFX permettant de charger un fichier de données au format .arff, d'entraîner un arbre de décision selon l’algorithme ID3, et de visualiser graphiquement la structure de l’arbre généré. Il s’agit d’un outil interactif destiné à l’apprentissage et à l’expérimentation autour des concepts de classification supervisée.

Fonctionnalités principales :

Interface graphique conviviale développée avec JavaFX

Chargement de fichiers .arff contenant les données à analyser

Implémentation de l’algorithme ID3 pour la construction de l’arbre

Visualisation de l’arbre de décision entraîné

Possibilité de tester de nouvelles instances à travers l’interface

Affichage des données (instances, attributs, classes) dans une vue dédiée

Réinitialisation de l’arbre pour expérimenter avec différents fichiers

Technologies utilisées :

Java (JDK 17 ou supérieur recommandé)

JavaFX (interface graphique)

FXML (définition des interfaces)

CSS (pour le style graphique)

Git (gestion de version)
